# Content of this repository

This is code that is supporting Gosselin et al. 2026 (TBA).

This repo consists of python notebooks and scripts to run the random forests analysis and to plot the variable importances as well as the partial dependence plots. The partial dependence plot scripts were executed on an HPC. 

Data sources can be found in the paper above.
